$ ssh -T git@github.com 测试ssh-key是否成功

$ git config --global user.name  "name"//你的GitHub登陆名

$ git config --global user.email "123@126.com"//你的GitHub注册邮箱

$ git remote add origin git@github.com:flora0103/example.git    //关联一个远程库命令， 

$ add .        （注：别忘记后面的.，此操作是把master文件夹下面的文件都添加进来）

$ git commit  -m  "提交信息"  （注：“提交信息”里面换成你需要，如“first commit”）

$ git push -u origin master   （注：此操作目的是把本地仓库push到github上面，此步骤需要你输入帐号和密码）

$ git clone 项目地址 （此处的项目地址）
